# ROX
Este proyecto esta orientado a brindar un servicio gratuito y libre para los usuarios del transporte público de Guatemala.

## Objetivos
- Democratizar el acceso a información de la red de transporte y crear las bases tecnologica para el sistema unificado de transporte público de Guatemala.
- Promoveer la creación de un sistema de transporte más eficiente y seguro.
- Incentivar el uso de transporte colectivo sobre el uso de vehiculos de uso personal.

## Funciones
- Planificador de viajes
- Mapa interactivo en tiempo real
- Visualización y categorización de opciones de transporte por ruta
- Tablas con acceso a la información de las diferentes rutas y opciones de transporte disponibles con información general.
